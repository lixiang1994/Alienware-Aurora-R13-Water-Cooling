## 前言

恰逢双十一 12代发布了 i9 12900K 非常给力, 于是琢磨是时候该换一套了.

首发的主板和CPU着实难抢, 也恰巧这时候外星人R13发布了, 外观实在长在我审美上了(我一直对ROG的杀马特灯不感冒)
